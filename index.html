Basic HTML Document Structure CheatSheet

1. Declaration (<!DOCTYPE html>):

The <!DOCTYPE> declaration defines the document type and version of HTML being used. In this case, html signifies HTML5, the latest version of HTML. This declaration ensures that the browser interprets the HTML code correctly.

2. HTML Element (<html lang="en">):

The <html> element is the root element of an HTML document. The lang attribute specifies the language of the document (in this case, English). This attribute aids accessibility and language-specific styling.

3. Head Section (<head>):

The <head> element contains meta information about the HTML document. This includes the title, character set, and other relevant metadata.

• Title (<titleYour> Title Here </title>):

The <title> element sets the title of the HTML document, which is displayed in the browser's title bar or tab. It provides a brief description of the page's content.

4. Body Section (<body>):

The <body> element represents the content of an HTML document. It contains all the elements that make up the document's structure.
